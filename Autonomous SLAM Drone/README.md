# Autonomous SLAM Drone

A visual preview of an autonomous SLAM drone project, including build photos and flight test videos.

This drone was built with custom calculations and a core hardware/software stack including an Intel RealSense D435, Raspberry Pi, ROS 2, Pixhawk, and an optical flow sensor as one of the main core components.
